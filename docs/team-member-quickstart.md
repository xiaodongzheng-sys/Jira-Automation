## Daily Use

1. Open Chrome and make sure BPMIS is logged in  
   打开 Chrome，确认 BPMIS 已登录
2. Open Terminal in the project folder  
   在项目目录打开 Terminal
3. Run:

```bash
./scripts/run_team_stack.sh start
```

4. Open [http://127.0.0.1:5000](http://127.0.0.1:5000)  
   打开 [http://127.0.0.1:5000](http://127.0.0.1:5000)
5. Click `Preview Eligible Rows`  
   点击 `Preview Eligible Rows`
6. If the preview looks correct, click `Run Ticket Creation`  
   如果预览没问题，再点击 `Run Ticket Creation`
